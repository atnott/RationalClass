# Rational-CPP

Проект с реализацией собственного класса для работы с рациональными числами и встроенным решателем квадратных уравнений.

## Особенности проекта
- **Класс `Rational`**:
    - Хранение чисел в виде пары «числитель/знаменатель».
    - Автоматическое сокращение дробей.
    - Перегрузка операторов арифметики и сравнения.
- **Решение уравнений**:
    - Функция для поиска корней квадратного уравнения $ax^2 + bx + c = 0$.

## Использование

### Создание рациональных чисел
```cpp
Rational r1(1, 2);  // 1/2
Rational r2(2, 4);  // Автоматически станет 1/2
Rational sum = r1 + r2; // Результат: 1/1
